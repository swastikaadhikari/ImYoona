Introduction


Web development deals with the design part and this is the first assignment given to us after joining college. So, basically we were expected to prepare atleast 6 web pages using HTML and CSS properties. The page are as index page which is also called as landing or first page, about page which should give the short introduction about ourselves including the reason why we aspire to be a developer. CV page contains the information about the experinces, or detailed information using flex properties. Social page include the link to the professional social pages we are connected to and contact page include a form and google map showing our address. The main objective of giving this assignment was to make us understand more about web development and too see our creativity as well. These web pages should be responsive as well and have UI design because it make the web site or pages look more attractive.
